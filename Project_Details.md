# Pizza Sales Dashboard Project

## Overview
This project analyzes pizza sales data using SQL and Excel to generate business insights and build an interactive dashboard.

## Data Source
- Data imported from SQL Server into Excel

## Workflow
1. Data cleaning using SQL
2. Data analysis using SQL queries
3. Data loading into Excel
4. Pivot tables creation
5. Dashboard building using charts and slicers

## Sheets Used

### 1. Pizza Sales
Raw dataset loaded from SQL

### 2. KPI
- Total Revenue: $817,860
- Total Orders: 21,350
- Total Pizzas Sold: 49,574
- Avg Order Value: $38.31
- Avg Pizzas per Order: 2.32

### 3. Trends
- Daily order trends (Column Chart)
- Hourly order trends (Line Chart)

### 4. Sales Analysis
- Sales by Category (Donut Chart)
- Sales by Size (Pie Chart)

### 5. Performance
- Top 5 Best Selling Pizzas
- Bottom 5 Worst Selling Pizzas

### 6. Dashboard
Final interactive dashboard combining all KPIs and visuals

## Key Insights
- Highest orders occur on weekends (Friday & Saturday)
- Peak hours: 12–1 PM and 5–8 PM
- Classic category contributes highest sales
- Large size pizzas generate maximum revenue

## Conclusion
The project demonstrates how SQL and Excel can be used together for business analysis.
